Firebase By Example

This page shows the following different Firebase features:

  • Hosting
  • Relatime Database
  • File Storage
  • Firestore Database
  • Authentication
  • Cloud Functions

Hosting

This site is hosted on firebase. You can use firebase hosting to host your SPA's and even full blown web application written in angular for exmaple.

Firestore Database

Using the form at the Realtime Database section, the information will also be uploaded to firestore. Use the button to perofrm a query that gets a document back by a given name.

Authentication

Firebase lets you manage users and help you with the authentication process. Several authentication providers are enabled by firebase including: Google, Facebook, Github, email/password etc. To enable a provider you have to manually choose it from the Firebase panel. In this part we can see an implementation of authentication using Google.

Cloud Functions

Firebase cloud function can be used to run backend code without dealing with servers and application servers. Click the echo button to trigger a deployed cloud function.

Place this url in your browser:

https://us-central1-fir-by-example.cloudfunctions.net/echoMessage?text=

Add some text after the '=' sign to echo this text back
